## How to Run
1. Unzip the folder
2. Set up cloud storage (see below)
3. Open index.html in any browser
4. Share the link — all users see the same centralized data!

## Install on Phone
1. Upload to GitHub Pages (free)
2. Open the URL on your phone
3. Tap Add to Home Screen
